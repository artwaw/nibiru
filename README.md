# nibiru

## Multiplatform (Windows x64, Linux x64, macOS) tool for astrological charts and basic interpretations of many planetary aspects

---

Technical details:

### Built with:
- Qt 5.12.0 (Windows: MinGW 7.4 x64)
- SQlite 3

### Libraries needed (Windows):
- Qt5Core.dll
- Qt5Gui.dll
- Qt5Sql.dll
- Qt5Network.dll
- Qt5Widgets.dll
- Qt5PrintSupport.dll
- Qt5Charts.dll
- Qt5Help.dll
- libwinpthread-1.dll
- libstdc++-6.dll
- libgcc_s_seh-1.dll

---

### Third party software used:

- SQLite dll for local db access https://sqlite.org/index.html
